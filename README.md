# scraped-cli

A powerful command-line web scraping tool that extracts structured data from websites using CSS selectors.

## Features

- 🚀 **Fast & Lightweight** - Built with axios and cheerio
- 🎯 **CSS Selector Support** - Use standard CSS selectors to target elements
- 📊 **Multiple Extraction Modes** - Single selector or mapped fields
- 💾 **JSON Output** - Get structured data in JSON format
- 📁 **File Export** - Save results directly to JSON files
- 🔧 **Custom Headers** - Set custom HTTP headers for requests
- ⏱️ **Timeout Control** - Configure request timeouts
- 🎨 **Attribute Extraction** - Extract text, HTML, or specific attributes
